SASS

Other than Foundation, other popular frameworks that use SASS include Compass, Bourbon and Suzy. It really is the most feature-rich pre-processing language. It is well-supported and has a large community. Since version 3, its main syntax is SCSS (.scss). It is more verbose than its previous SASS syntax (still supported). But the key advantage of SCSS is that all CSS3 syntax are also valid. This makes development and transition easier. Official site | On Github

Variables

$color: #000000;
$size:1em;
$boolean:true
$font:"Helvetica Neue", "Helvetica", Helvetica, Arial, sans-serif;
$calculation: darken($color, 15%);
$cursor: "crosshair"

font: $size $font;

Compiles to:

font: 1em "Helvetica Neue", "Helvetica", Helvetica, Arial, sans-serif;

$style: solid !default;
The !default is special to Sass variables. It sets, well, a default value. Foundation uses it quite extensively to better organize user style overrides.

Nesting

div {
 h1 {
   padding: 2em;

    span {
      font-weight: normal;
    }

 }
 a {
   display: none;
 }
}
Sass supports nesting properties of CSS namespaced properties
.myfont {
    font: {
    family: Myraid;
    size: 10px;
    weight: normal;
   }
}

Compiles to:
div h1 { padding: 2em; }
div h1 span { font-weight: normal; }
div a { display: none; }
.myfont {
 font-family: Myraid;
 font-size: 10px;
 font-weight: normal;
}

Imports

@import "lib";/* file imported and processed */

.class {
   @import "lib";/* file imported to class */
}

@import "file1", "file2";/* both file imported and processed */

To output CSS @import, simply write it in CSS syntax

@import url("my.css");

Compiles to:
@import url("my.css");

}

Compiles to:
.mixin, .o {
 color: #000;
 height: 100px;
}
.o {
 width: 100%;
}

Mixin

@mixin mybuttonHover ($bgdcolor:#e6e6e6, $fontcolor:#333333) {
 background-color: $bgdcolor;
 color: $fontcolor;
 text-decoration:none;
 background-position: 0 -99px;
}
a.mybutton:hover {
 @include mybuttonHover();
}
a.mybutton1:hover {
 @include mybuttonHover(#990000);
}
a.mybutton2:hover {
 @include mybuttonHover($fontcolor:#000000);
}
@mixin box-shadow($shadows...) {
 box-shadow: $shadows;
 -moz-box-shadow: $shadows;
 -webkit-box-shadow: $shadows;
}
.myshadow {
 @include box-shadow(1px 2px 1px #000);
}

Compiles to:
a.mybutton:hover {
 background-color: #e6e6e6;
 color: #333333;
 text-decoration: none;
 background-position: 0 -99px;
}
a.mybutton1:hover {
 background-color: #990000;
 color: #333333;
 text-decoration: none;
 background-position: 0 -99px;
}
a.mybutton2:hover {
 background-color: #e6e6e6;
 color: black; /*compiled from hex */
 text-decoration: none;
 background-position: 0 -99px;
}
.myshadow {
 box-shadow: 1px 2px 1px black;
 -moz-box-shadow: 1px 2px 1px black;
 -webkit-box-shadow: 1px 2px 1px black;
}

Functions/Operation

Full reference here

Math calculation such as floor($var). Color manipulation such as lighten($color, 10%), convert HEX to HSL/RGBA, check type, or mix colors together. Surprisingly, Sass has only a few built-in math functions. But You can also write your own function directives.

$c: #990033;
.box {
 border-color: rgba($c, 0);
 background: mix(#f00, #00f, 25%);
}


@function power ($x, $n) {
 $ret: 1;

 @for $i from 1 through $n {
    $ret: $ret * $x;
 } 

 @return $ret;
}

//calculate line-height based on pearsonified.com golden ratio
@mixin lineht($fontsize, $width, $gr: 1.61803) {
  line-height: ceil(($gr - (1/(2*$gr))*(1 -($width/(power($fontsize*$gr,2))))) * $fontsize)+px;
}
p {
 /* width: 1em + 2px; invalid operation*/
 @include lineht(12,600);
}
(Power function from Sass-Math project)

Compiles to:

.box {
 border-color: rgba(153, 0, 51, 0);
 background: #3f00bf;
}

p {
 line-height: 22px;
}Sass, surprisingly, has fewer math operations

Conditionals/Loops

Full reference here

Pretty easy to understand as it uses programming syntax: @if, @else, @for, @each, @while.

@mixin BeforeClass($fp, $fn) {
@if $fp != null {
[class^="social-icons-#{$fp}-"]:before,
 [class*=" social-icons-#{$fp}-"]:before {
 font-family: $fn;
}
} @else {
 font-family: $fn;
 font-size: 1em;
 }
}
